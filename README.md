## The Pricing Model

Every AAS quote is generated by an 8-variable multivariable cost function:

**C(f, d, h, φ, e, m, r, κ) → Optimized Price**

Where:
- f = fuel cost
- d = route distance
- h = labor hours
- φ = crew fatigue factor (declining efficiency over time)
- e = equipment depreciation
- m = materials cost
- r = disposal fees
- κ = seasonal demand coefficient

The model uses partial derivative optimization to find the price that maximizes margin while remaining competitive. It has been back-tested against all 44 completed jobs.
